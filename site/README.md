# GitHub Pages site

Static landing pages for **https://org-event.github.io/OpenPolySphere/** (English + Russian).

| Path | Purpose |
|------|---------|
| `index.html` | English homepage |
| `ru/index.html` | Russian homepage |
| `css/site.css` | Styles |
| `img/` | Favicon and site assets |
| `readme-logo.png` | Header logo |

Deployed by [`.github/workflows/pages.yml`](../.github/workflows/pages.yml) on push to `main`.

Local preview: open `site/index.html` in a browser (relative paths).

Project documentation (Markdown) lives in [`docs/`](../docs/) — not published to Pages.
