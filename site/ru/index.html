Переводчик звонков в реальном времени

Переводит обе стороны разговора на лету: вы говорите на своём языке, собеседник слышит на своём, и наоборот.

macOS 14+ Windows и Linux (CI-сборки) 29 языков MIT

Как это работает: Микрофон → распознавание речи → перевод → синтез речи → обратно в звонок. По умолчанию локальный режим Whisper + Opus-MT; облачные бэкенды опционально.

Быстрый старт

git clone https://github.com/org-event/OpenPolySphere.git
cd OpenPolySphere
./scripts/bootstrap
cargo run --release -p translator -- setup   # первый раз: модели
cargo run --release -p translator            # открыть http://127.0.0.1:5050

Для звонка и веб-интерфейса используйте Google Chrome.
